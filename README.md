# subdscan
Subdomain fast search tool

## Usage
```
# params
-d string
      Set domain scan like '-d ya.ru'
-t int
      Set parallelism like '-t 50' (default 10)
-w string
      Set subdomains dictionary file like '-w wordlist.txt' (default "wordlist.txt")

#run
./subdscan -d somesite.ru
```
